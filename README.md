# TCGA-KIRC Multi-Omics Survival Signature

This project explores clear cell renal cell carcinoma (ccRCC) using a real multi-omics strategy based on TCGA-KIRC data.

The objective is to identify clinically relevant molecular subtypes by integrating transcriptomic, proteomic, and survival information.

## Data layers
- RNA-seq expression
- RPPA proteomics
- Clinical and survival data

## Main analyses
- Patient matching across omics
- Integrative clustering
- Survival analysis
- Differential expression
- Functional enrichment
- Biological interpretation of concordant signals

## Clinical relevance
Rather than building a generic cancer pipeline, this project aims to connect molecular heterogeneity with prognosis in renal oncology.
